Introducción a Vectores en Física

¿Qué son los Vectores y Escalares?

Escalar: Una cantidad que solo tiene magnitud (un número). Ejemplos: masa (5 kg), temperatura (20 °C), tiempo (10 s).

Vector: Una cantidad con magnitud y dirección. Se representa con componentes en un sistema de coordenadas (ejemplo: (3, 4) en 2D) o con una flecha en gráficos. Ejemplos: velocidad (10 m/s al norte), fuerza (20 N a 30°).

Los vectores son esenciales en física para describir fenómenos como movimiento, fuerzas y campos, ya que la dirección es tan importante como la magnitud.

Operaciones con Vectores

  • Suma/Resta: Se suman o restan las componentes correspondientes. Ejemplo: Si A = (2, 3) y B = (1, -1), entonces A + B = (2+1, 3-1) = (3, 2).
  • Producto Escalar: Resulta en un escalar. A · B = |A||B|cos(θ) o A · B = AₓBₓ + AᵧBᵧ (en 2D). Mide la proyección de un vector sobre otro.
  • Producto Vectorial (3D): Resulta en un vector perpendicular a los dos vectores originales. A × B = (AᵧBₓ - AₓBᵧ, AₓBₓ - AₓBₓ, AₓBᵧ - AᵧBₓ). Útil para calcular momentos o áreas.

Vectores en 2D y 3D

2D: Un vector en 2D tiene dos componentes (x, y). Ejemplo: (4, 5) representa un vector con magnitud √(4² + 5²) = √41 y dirección θ = atan(5/4).

3D: Un vector en 3D tiene tres componentes (x, y, z). Ejemplo: (1, 2, 3). La magnitud es √(x² + y² + z²), y la dirección requiere ángulos en tres dimensiones.

En física, los vectores 2D se usan en mecánica plana (movimiento en un plano), mientras que los vectores 3D son comunes en dinámica espacial o electromagnetismo.

Ejemplos con Geometría Analítica

Ejemplo 1: Suma de Vectores

Dados A = (3, 2) y B = (-1, 4), calcular A + B.

Solución: A + B = (3 + (-1), 2 + 4) = (2, 6).

Ejemplo 2: Producto Escalar

Dados A = (2, 3) y B = (1, -1), calcular A · B.

Solución: A · B = (2)(1) + (3)(-1) = 2 - 3 = -1.

Ejemplo 3: Producto Vectorial (3D)

Dados A = (1, 0, 2) y B = (0, 1, -1), calcular A × B.

Solución: A × B = ((0)(-1) - (2)(1), (2)(0) - (1)(-1), (1)(1) - (0)(0)) = (-2, 1, 1).

Ejemplo 4: Distancia entre Puntos

Calcular la distancia entre P(1, 2) y Q(4, 6) usando vectores.

Solución: Vector PQ = (4-1, 6-2) = (3, 4). Magnitud = √(3² + 4²) = √25 = 5.
