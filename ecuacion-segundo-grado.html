Ecuación de Segundo Grado

$$ax^2 + bx + c = 0$$
Forma general de la ecuación cuadrática

Significado de los coeficientes

$$ax^2 + bx + c = 0$$
Donde \(a\), \(b\) y \(c\) son números reales, y \(a \neq 0\)

Fórmula General (Resolvente)

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Esta fórmula permite encontrar las soluciones de cualquier ecuación cuadrática.

El Discriminante

$$\Delta = b^2 - 4ac$$
El discriminante determina la naturaleza de las soluciones.

Casos según el Discriminante

  • Si \(\Delta > 0\): dos soluciones reales distintas
  • Si \(\Delta = 0\): una solución real doble
  • Si \(\Delta < 0\): dos soluciones complejas conjugadas

Ejemplo

$$2x^2 - 4x - 6 = 0$$
Resolvamos esta ecuación paso a paso.

Identificamos los coeficientes

$$a = 2 \qquad b = -4 \qquad c = -6$$
Comparamos con la forma general \(ax^2 + bx + c = 0\).

Cálculo del Discriminante

$$\Delta = (-4)^2 - 4(2)(-6) = 16 + 48 = 64$$
Como \(\Delta = 64 > 0\), hay dos soluciones reales distintas.

Sustitución en la Fórmula

$$x = \frac{-(-4) \pm \sqrt{64}}{2(2)} = \frac{4 \pm 8}{4}$$

Soluciones Reales

$$x_1 = \frac{4 + 8}{4} = 3 \qquad\qquad x_2 = \frac{4 - 8}{4} = -1$$
Las soluciones son \(x_1 = 3\) y \(x_2 = -1\)
