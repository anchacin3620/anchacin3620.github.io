Introducción: Películas Delgadas y Simulación

Contexto Físico

  • Las películas delgadas de Cromo (Cr) e Hierro (Fe) son cruciales para recubrimientos protectores y dispositivos magnéticos.
  • El sustrato de Magnesio (001) (estructura hexagonal compacta) presenta desafíos de corrosión que requieren recubrimientos optimizados.
  • La dinámica a escala atómica (adsorción y difusión) determina la morfología final.

Simulación Estocástica

El método Monte Carlo Cinético (KMC) permite modelar procesos fuera del equilibrio que son imposibles de resolver con ecuaciones deterministas simples.

"Se utiliza la ecuación maestra probabilística como base teórica para derivar tasas de transición de adsorción y difusión."

Metodología: Algoritmo KMC

Modo Balístico

En esta simulación, se utiliza el modo de deposición balístico:

  • Energía promedio de partículas: 5.0 eV.
  • Simula deposición a alta energía.
  • Promueve crecimiento vertical y formación de columnas.

Implementación

Código desarrollado en Fortran 90 para el motor de simulación y Python para el análisis post-simulación (mapas 2D/3D, métricas).

Ciclo del Algoritmo

1. Inicializar Rejilla (Sustrato Mg)
↓
2. Selección de Evento (Deposición/Difusión)
↓
3. Actualización de Sistema (Átomos Cr/Fe)

Parámetros del Sistema

Propiedades físicas intrínsecas utilizadas para definir las tasas de transición en el modelo KMC.

Cromo (Cr)

  • Adhesión (Sticking): 0.98 (Alta)
  • Energía de Difusión (Ed): 0.01 - 0.25 eV (Baja)
  • Comportamiento: Menor movilidad superficial, estructuras más rígidas y fragmentadas.

Hierro (Fe)

  • Adhesión (Sticking): 0.95
  • Energía de Difusión (Ed): 0.5 - 1.0 eV (Alta)
  • Comportamiento: Mayor movilidad superficial, forma picos pronunciados y valles profundos.

Resultados: Modo Balístico

Comparación métrica obtenida tras el análisis de las simulaciones.

Conclusiones

  • Validación del Modelo: El código en Fortran 90 basado en KMC demostró alta precisión para modelar deposición estocástica.
  • Patrones de Crecimiento: Aunque Fe y Cr tienen valores de rugosidad similares, sus morfologías son distintas. Fe es más homogéneo en transiciones, mientras que Cr es más fragmentado y abrupto.
  • Eficiencia Computacional: El uso combinado de Fortran (cálculo numérico pesado) y Python (visualización) resultó óptimo para este estudio.
  • Modo Balístico: Resultó clave para alcanzar espesores mayores y definir nanoestructuras columnares.
